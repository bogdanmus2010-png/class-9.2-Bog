# class-9.2-Bog
homework
1. Концепція продукту
Тип додатку: Чат-бот (Telegram)
Основна ідея: Бот допомагає користувачам швидко виконувати математичні розрахунки прямо в Telegram без потреби відкривати калькулятор.
2. Основний функціонал (Features)
[ ] Приймати текст від користувача
[ ] Робити математичний розрахунок
[ ] Виводити результат
[ ] Показувати список команд
[ ] Завершувати роботу за командою
3. Структура даних
user_input - тип str. Потрібна для збереження повідомлення користувача.
result - тип float. Потрібна для збереження результату обчислення.
is_active - тип bool. Потрібна для контролю роботи бота.
command - тип str. Потрібна для збереження команди користувача.
numbers - тип list. Потрібна для чисел, які використовуються у розрахунках.
